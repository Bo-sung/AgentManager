# design — UI 디자인 레퍼런스 (HTML)

여기에 Claude로 만든 **UI 디자인 HTML/CSS/에셋**을 넣으세요.
이 폴더는 **XAML 포팅의 시각·구조 명세**일 뿐, WPF 앱에 컴파일되지 않습니다(방식 A: 네이티브 XAML 재구현).

## 넣을 것
```
design/
  index.html        ← 메인 디자인 (필수)
  *.css             ← 분리된 스타일이 있으면
  assets/           ← 아이콘/이미지/폰트 등 (있으면)
  screens/          ← 참고 스크린샷(있으면, png)
```

## 넣은 뒤
- 경로를 알려주시면 이 디자인을 읽고 `src/AgentManager/`(WPF)에 **XAML로 재구현**합니다.
- 방금 만든 Core(`AgentSession`/정규화 이벤트/번역 토글)에 바인딩해 빌드·실행까지 맞춥니다.

## 참고
- HTML/CSS의 레이아웃·색·구성요소는 충실히 재현하되, 렌더 모델 차이로 **100% 픽셀 동일은 아님**(그림자/폰트 미세차는 WPF 관용구로 근사).
- 외부 폰트/아이콘을 쓰면 라이선스 확인 후 앱에 동봉하거나 시스템/대체 리소스로 매핑합니다.
